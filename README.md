# CSCI 221 - Computer Science II

**Spring 2026 | AUPP/FHSU**  
**Textbook:** *Problem Solving with C++* by Walter Savitch

This repository organizes all coursework by textbook chapter, including labs, practice problems, lecture notes, code snippets, and test materials.

## Repository Structure

### Chapter Organization
All work is organized under `/chapters/` by chapter number:

- [Chapter 9: Pointers and Dynamic Arrays](chapters/ch09)
- [Chapter 10: Defining Classes](chapters/ch10)
- [Chapter 11: Friends, Overloaded Operators, Arrays in Classes](chapters/ch11)
- [Chapter 12: Separate Compilation and Namespaces](chapters/ch12)
- [Chapter 13: Pointers and Linked Lists](chapters/ch13)
- [Chapter 15: Inheritance](chapters/ch15)
- [Chapter 16: Exception Handling](chapters/ch16)
- [Chapter 17: Templates](chapters/ch17)
- [Chapter 18: Standard Template Library and C++11](chapters/ch18)

### Subfolder Types

Each chapter contains the following subdirectories:

- **labs/** - Official lab assignments and submissions
- **practice/** - Extra practice problems and exercises
- **notes/** - Lecture notes, study guides, and references
- **snippets/** - Code examples and demonstrations
- **tests/** - Quiz and exam review materials

## How to Use This Repository

1. Navigate to the relevant chapter folder
2. Check the chapter's README.md for topics and checklist
3. Add your work to the appropriate subfolder (labs, practice, etc.)
4. Use the checklist to track your progress through each chapter
